# Tamarin in 90 Minutes
Material for a Tamarin Tutorial in 90 Minutes, for the _2026 Journées du GT MFS_ in Luz-Saint-Sauveur, France

Please install the Tamarin-prover before the start of the tutorial. 
You can find information on how to install it in the [Installation Guide](https://github.com/jdreier/tamarin-90min/blob/main/installation/InstallationGuide.md). 

## Material

* The slides from the lecture can be found in the `slides` folder.
* The folder `exercises` contain exercises for the practical part.
* The folder `naxos` contain exercises on the NAXOS protocol, continuing the example from the lecture.
* This [cheatsheet](https://github.com/felixlinker/tamarin-workshop/blob/main/cheatsheet.md) from another Tamarin workshop (by Felix Linker) is a good summary of Tamarin's syntax.
* In case you have any doubts or questions on the syntax and usage of Tamarin, the [Tamarin book](https://tamarin-prover.com/book/index.html) and the [Tamarin manual](https://tamarin-prover.com/manual/master/book/001_introduction.html) are available online and are great sources of information.
* Cas Cremers created a helpful illustration (`dependencygraph.pdf`) of Tamarin's GUI output.
